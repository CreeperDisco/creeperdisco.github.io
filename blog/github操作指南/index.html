4. 界面汉化设置

  • 网页端汉化：

    1. 登录GitHub后点击右上角头像
    2. 选择Settings > Appearance
    3. 在Language选项中选择中文(简体)或使用浏览器翻译功能
    4. 保存设置

  • GitHub Desktop汉化：

    1. 打开GitHub Desktop
    2. 点击顶部菜单File > Options
    3. 在General标签页中找到Language下拉菜单
    4. 选择中文(简体)后重启客户端
